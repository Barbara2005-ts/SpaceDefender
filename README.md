# Космический Защитник

2D аркада, разработанная на Godot 4 для курсовой работы.

## Описание
Игрок управляет космическим кораблем, защищаясь от волн врагов. 

## Особенности
- Управление кораблем (WASD/Стрелки)
- Система стрельбы с перезарядкой
- Враги с автоматическим спавном
- Система здоровья и очков
- Полнофункциональный интерфейс

## Управление
- **WASD/Стрелки** - движение
- **Пробел** - стрельба

## Сборка
1. Скачайте Godot 4.x
2. Клонируйте репозиторий
3. Откройте проект в Godot
4. Запустите сцену `Main.tscn`
